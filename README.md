# Not Dungeon v3

A simple, customizable AI-powered storytelling app inspired by AI Dungeon. This is a personal project focused on flexibility, extensibility, and experimentation with different AI backends. A never-ending obsession of mine to recreate.

## Features

- 🔌 **OpenAI-Compatible API Support**
  Connect to any API that follows the OpenAI format (local or hosted).

- 🧩 **Scriptable User Flow**
  Easily customize how the app behaves — from input handling to story progression.

- 🛠️ **Highly Customizable**
  Designed to be flexible so you can tweak prompts, logic, and interactions.

- 👤 **Personal Project**
  Built for learning, experimentation, and fun.

## Getting Started

```bash
git clone <your-repo-url>

cd <project-folder>
npm install  # or your preferred package manager

# Run the app
npm run dev
```

## Usage

1. Start the app
2. Enter a prompt or scenario
3. Let the AI generate the story
4. Continue interacting to shape your narrative

## Customization

You can modify:

- Prompt templates
- Conversation flow
- Response handling logic
- UI/UX behavior

The project is designed to make experimentation easy.
